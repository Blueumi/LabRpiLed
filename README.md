# LabRpiLed
라즈베리파이를 이용한 Rgb Led Matrix 제어 학습

## 라즈베리파이 실습을 위한 프로그램 다운로드

* Raspbian: 라즈베리파이 OS
* Etcher: 이미지를 (Micro)SD 카드에 굽는 프로그램
* TeraTerm: 라즈베리파이와 터미널로 연결하는 프로그램
  * MAC에서는 자체 탑재된 시리얼 프로그램 이용
* Visual Studio Code (VSCode): 프로그램 개발 툴
* FileZilla: 파일 전송 프로그램
