# Raspbian Update and Upgrade
> Upgrade시 소요시간이 제법 길기 때문에 교육시에는 실행하지 말 것

<pre>
<code>pi@raspberrypi:~ $ sudo apt-get update
pi@raspberrypi:~ $ sudo apt-get upgrade</code>
</pre>

> 필요시 설치
<pre><code>sudo apt-get install rpi.gpio</code></pre>
